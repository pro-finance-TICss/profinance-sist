# proinance-sist

Proyectos software Pro-Finance. La documentación de cada proyecto está en su respectivo archivo README de su respectiva carpeta.

## Ejecución de los proyectos

1.  Instalar dependencias:

    ```bash
    npm install
    ```
Se tiene que iniciar el servidor de desarrollo indicando el proyecto:

2.  Iniciar servidor de desarrollo de la landing:

    ```bash
    npm run landing:dev
    ```

2.  Iniciar servidor de desarrollo de la app:

    ```bash
    npm run app:dev
    ```
